web 图片单位简介

Share

计算机图形上有上百种不同的文件格式，因此会有许多单调而乏味的缩写：gif、jpeg、png、tiff、bem 等。而我们在前端开发中，网络中使用到的图形格式会少一些，常见的图形格式有：gif、jpeg、png 以及 svg。

gif、jpeg 和 png 这三种格式都有很好的压缩效果，经过计算机巧妙的处理之后图形文件的大小会得到压缩，从而在互联网中传输的速度更快。我们可以根据自己的需求选择在网页中使用哪一种格式。

层叠和继承

Share

层叠和继承，是 css 两大核心概念，当样式表越写越复杂，你可能会开始考虑为啥页面中的元素会显示成那样？有时样式又会冲突，也就是一个元素被多个规则声明了相同的 css 属性。幸好 css 会使用一种机制处理掉冲突，这种机制就叫做“层叠”，用于管控样式间相互作用的方式，出现冲突时判断哪个样式的优先级高，就执行哪一种。
