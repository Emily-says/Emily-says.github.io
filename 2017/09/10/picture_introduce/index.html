web 图片单位简介

Share

计算机图形上有上百种不同的文件格式，因此会有许多单调而乏味的缩写：gif、jpeg、png、tiff、bem 等。而我们在前端开发中，网络中使用到的图形格式会少一些，常见的图形格式有：gif、jpeg、png 以及 svg。

gif、jpeg 和 png 这三种格式都有很好的压缩效果，经过计算机巧妙的处理之后图形文件的大小会得到压缩，从而在互联网中传输的速度更快。我们可以根据自己的需求选择在网页中使用哪一种格式。

GIF

GIF (graphics interchange format) 格式对于纯色图形的压缩效果很好，适用于徽标、文字，以及简单的横幅等等。GIF 格式还可以让元素颜色透明，并且包含简单的动画。

不过它最多只能包含256种颜色，通常会丢失一些色彩，因此，使用数码相机拍摄的图像如果使用 GIF 作为格式，那么显示效果就不会太好了。

JPEG

JPEG (joint photographic experts group) 图形弥补了 GIF 格式的缺点，它能包含百万种不同色彩，特别适合于照片。此外， JPEG 对多彩图像的压缩效果也比 GIF 好很多，因为 JPEG 采用的压缩算法会考虑人眼对相近颜色的感知。很多相机都使用 JPEG 格式来保存图片。图形处理软件保存 JPEG 文件时会做复杂的颜色分析，减少准确表现图像所需的数据量。不过 JPEG 有个缺点就是压缩后文字和大片纯色会出现斑点。

PNG

PNG (portable network graphics) 格式身兼 GIF 和 JPEG 的优点，不过要根据场合来选用合适的 PNG 版本。

PNG8

PNG8 基本上能替代 GIF ，与 GIF 一样， PNG8 最多包含256色，而且只能有一种颜色是透明的。不过它的压缩率通常比 GIF 高一些，因此相同的图像保存为 PNG8 要比 GIF 下载速度稍快一些。也就是说应该优先选择 PNG8。

PNG24 与 PNG32

它俩基本算是 PNG24 + Alpha 透明通道两种格式比 JPEG 包含的颜色更多，因此图像品质无损，也就是说保存为 PNG24 或者 PNG32 格式的照片品质会比 JPEG 要好。不过在拥抱 PNG 格式之前要知道 JPEG 的品质也很好，而且文件大小比 PNG24 和 PNG32 都要小。一般来讲 JPEG 更适合于保存照片和其他包含大量颜色的图像。

PNG32 格式又个其他格式都没有的特性，256色透明，也叫做 Alpha 通道透明。可以透过图形里的投影看到网页背景。也可以实现半透明效果。

SVG

SVG (scalable vector graphic) 与前面几个都不同，它不是位图，不是由一个个一排排极小的像素组成的，而是用一系列数学指令说明怎样在屏幕上绘制图像。SVG 和其他矢量格式不适用于保存图片和图画，而适合于保存简单的图形，比如徽标或图标。它的优点在于文件体积小，而且能在保持清晰度的前提下缩放任何尺寸。

除了 IE8 以外所有主流现代浏览器都能使用 SVG 格式的图片，然后像其他图形文件那样引用。
