纯 css 实现点击切换轮播效果

Share

在网页效果中，若是想实现 点击切换图片 样式，在你的印象中是不是只能利用牛逼哄哄的 js 才能实现？这篇博客就将介绍一种 “半智能” 的点击切换效果

你可以从 这里 下载源代码

首先用图解的方式看看完成后的效果图

⬆️ 这个图有点失败，一股子淘宝爆款味儿 - -

首先来看看代码整体结构

由 main 包含块包含了里面所有内容。cont 包含块包含了大图组 imgs 和小图组 nav 图片的排版布局方式就不多说了都是通过 position 定位的方式定在指定点

整篇代码中的关键点也是启发点在于 “锚点”

我们已经知道锚点其实就是通过超链接 a 标记中的 href 属性去链接一个具有唯一性的 id 值。在小图上设置 a 标记 href 属性分别链接上大图组上的每一张图

  • href="banner1" 关联 id="banner1"
  • href="banner2" 关联 id="banner2"
  • …

如果想在大图上添加一些介绍文字，同样也可以使用蒙版遮罩效果，定位在指定点

介样一来就完成了整个效果，利用锚点实现点击切换。不过在前面我说过这只是一个 “半智能” 的效果，这是因为如果网页存在 header 并且这个 banner 后还有内容的话，那么在点击后，图片就会升到最顶端，导致用户想再使用 header 时的不友好体验 ⬇️

事实是 ⬇️

造成酱紫的原因就在于 a 标记的锚点会使页面返回到最 顶 部 🆙

当然了我们在实际开发时肯定还是会选择 js 的方式，不过光论 demo 的话，这个效果不也是棒棒的么ヽ(•̀ω•́ )ゝ
