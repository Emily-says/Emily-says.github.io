首先 结构 创建

<div class="drawer">
    <ul>
        <li></li>
        <li></li>
        <li></li>
        <li></li>
        <li></li>
        <li></li>
    </ul>
</div>

加上样式，样式的原理是 后一个元素 覆盖上 前一个兄弟元素 宽度的一半

图片的实现效果是 sprite 技术 ／ 雪碧图

不要再一脸懵哔——问我雪碧图如何实现的了好吗？在课上我是不是说过 “把大象装进冰箱需要几步”？

每个人都告诉我需要3步

  • 把冰箱门打开（建视口）
  • 把大象放进去（插背景图）
  • 把冰箱门关上（改变背景图位置）
.drawer{
     padding: 12px;
     overflow-x: hidden;
 }
.drawer ul{
    position: relative;
    height: 100px;
    overflow-x: hidden;
}
.drawer li{
    width: 220px;
    height: 100px;
    background: url("img/menu-button.jpg");
    position: absolute;
    transition: left ease-in .6s;     //取值只取过 all ?有疑惑？看到最底下去
}
.drawer li:nth-child(1){
    top: 0;
    left: 0;
}
.drawer li:nth-child(2){
    background-position: -220px 0;
    top: 0;
    left: 130px;
}
.drawer li:nth-child(3){
    background-position: -440px 0;
    top: 0;
    left: 260px;
}
.drawer li:nth-child(4){

ok 联动效果的关键点来了，关键之处在于👉 ~ 选择器，它被解释为：

E ~ F 匹配任何在 E 元素之后的同级 F 元素

所以现在选择器需要 当 移入 第一个 li 时，后面的谁谁谁 怎样怎样，于是就有了以下

.drawer li:first-child:hover~li:nth-child(2){
    left: 220px;
}
.drawer li:first-child:hover~li:nth-child(3){
    left: 330px;
}
.drawer li:first-child:hover~li:nth-child(4){
    left: 440px;
}
.drawer li:first-child:hover~li:nth-child(5){
    left: 550px;
}
.drawer li:first-child:hover~li:nth-child(6){
    left: 660px;
}

选择完移入第一个，后面的兄弟级的联动效果以后，再为以后的每个 li 设置 当移入时，自己如何变化 。变化过程不仅移动了元素位置，还移动了背景图的位置，所以不仅 position 的值在改变， background-position 也在变

.drawer li:first-child:hover{
    background-position: 0 -100px;
}
.drawer li:nth-child(2):hover{
    left: 40px;
    background-position: -220px -100px;
}
.drawer li:nth-child(3):hover{
    left: 170px;
    background-position: -440px -100px;
}
.drawer li:nth-child(4):hover{
    left: 300px;
    background-position: -660px -100px;
}
.drawer li:nth-child(5):hover{
    left: 440px;
    background-position: -880px -100px;
}
.drawer li:nth-child(6):hover{
    left: 570px;
    background-position: -1100px -100px;
}

最后加上 向上移动 的效果，增加 “抖动” 感

.drawer li:hover{
    cursor: pointer;
    top: -2px;
}

在上面的代码中，出现了 transition 属性中的 left 值

.drawer li{
    ...
    transition: left ease-in .6s;
}

我们已经知道 li 在移入时位置 (position) 在变，背景图位置 (background-position) 也在变，
如果取值为 all 表示 position 和 background-position 同时经过一段时间在渐变，那么就能见到背景图的改变轨迹了，而我们只需要 position 具有渐变效果，你背景图 “咻——” 地一下变个位置就好咯 ╮（╯＿╰）╭
