4. border-radius



我们已经知道 border-radius: 10px; 可以为元素添加圆角效果



或者是 border-radius: 50px 10px 50px 10px;



也许你还没见过这样的 border-radius: 50% 50% 50% 50% / 60% 60% 40% 40%;



斜线分隔符 / 前面的值分别表示

  • 左上角圆角横向半径值
  • 右上角圆角横向半径值
  • 右下角圆角横向半径值
  • 左下角圆角横向半径值

/ 分隔符后面的值分别表示

  • 左上角圆角纵向半径值
  • 右上角圆角纵向半径值
  • 右下角圆角纵向半径值
  • 左下角圆角纵向半径值

4.2 制作半圆



制作半圆的技巧

  • 制作上／下半圆，元素的宽是高的2倍，圆角半径为元素的高度值
  • 制作左／右半圆，元素的高是宽的2倍，圆角半径是元素的宽度值

4.3 制作扇形


遵循 “3同1不同” 原则，“三同” 指元素宽度、高度、圆角半径值相同，“一不同” 指圆角位置不同

4.4 制作椭圆


制作椭圆的技巧：

  • 水平椭圆：元素宽度是高度的2倍，border-radius 的水平半径等＝元素宽度，垂直半径＝元素高度
  • 垂直椭圆：相反
